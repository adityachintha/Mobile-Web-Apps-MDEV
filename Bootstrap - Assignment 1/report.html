Report

I have used head, title, meta, script, style and link to structure my layout for the website. Also, i have used tags like body, nav, header, main, section, h1 tags etc, divs and img tags to organize my html document and add either components or data. Lastly, i have used formatting tags like strong, b, aria which is benefecial for search engines and screen readers.

I chose components as follows-

  • Carousel - To show example images in home page.
  • Cards with Customization - to show cards and sample data inside it on home page.
  • Navbar - I chose the navbar to display the navigation for the project.
  • Buttons - Used different types of button in about page like primary, secondary and outlined to showcase the button groups and also follow gestalt principles of similarity.
  • Cards - to show the card header and description in the services page and addded customization
